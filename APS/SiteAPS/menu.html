Proposta do Trabalho:

A proposta se baseia em criptografar mensagens para poder gerar uma certa segurança de que a mensagem não será interceptada e vista por nenhum outro tipo de pessoa que não seja o destinatário.
A criptografia em si se baseia na própria Cifra de César onde podemos substituir os valores de cada letra e número para poder formar uma mensagem de maneira criptografada. Para poder escriptografar, deve-se possuir uma chave especifica para cada mensagem. onde no momento existem um limite de 93 chaves possíveis, o que consequentemente torna 93 possiveis textos para serem de certa forma "interceptados".

Sobre o Código:

O código em si foi produzido em Python na qual se trata da linguagem de programação ensinada no curso além de ser uma linguagem de alto nível na qual exige menos trabalho com questões de tipagem e de sintaxe. Se trata de uma linguagem forte e de utilidade prática. O código está disponível na aba própria onde está localizado ao lado do Início. Para mais informações sobre o funcionamento e a produção do código, basta abrir a aba de Documentação onde possui todas as informações possíveis.
